# jk_bms
Lib for communicating with the jk bms  
Credit goes to @theMossCode  

## Attempt
The attempt is to have this library hardware independent

## modbus_rtu dependency
This repo has a dependency to the modbus_rtu repo (also done by @theMossCode)  
I've added it as a submodule though as it can be useful in other applications  
You can use symlinks to the submodules to develop on only one submodule
